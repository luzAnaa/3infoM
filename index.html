Versionamento

O Git é um software que permite realizar versões de um projeto. O GitHub Desktop é um software que faz a ligação entre VS Code, Git e GitHub.

O GitHub Desktop permite fazer a sincronia entre o repositório local e o repositório remoto.

Commit

É o processo de confirmação de uma versão, ou seja, a cada alteração é possível confirmar a alteração

Push

É o processo de envio do commit para o repositório remoto, ou seja, é a sincronização entre o repositório local e o remoto.
